DESCRIPTIONS OF MAWEB ROLES

THE FOLLOWING PERMISSIONS ARE AVAILABLE IN MAWEB

Permission Description
CanDeleteData The user can delete data from dissemination database. (not impemented yet)
CanDeleteStructuralMetadata The user can delete structures (Dataflows, Codelists, etc) which are accessible from the Browse & Download screen.
CanIgnoreProductionFlag The user can ignore production flag attribute when requesting Dataflows or Data (not impemented yet)
CanImportData The user can import the data to dissemination database. (not impemented yet)
CanImportStructures The user can view and interact with the Import from File & Import from Registry screens (using the available saved registries).
CanModifyStoreSettings The user can view the admin menu and configure the connection details of: authdb, mapping stores, dissemination database connections, manage registries, configure the proxy and add/edit/remove users. Also, he can save the default settings for structures download.
CanPerformInternalMappingConfig The user can view and interact with the the Datasets screen (create and remove datasets, save dataset columns etc.). Furthermore, he can view and interact with the Mapping screen, save mappings and transcodings and finally view mapped data. The user can only view/edit mapping sets which reference Dataflows that he has access to.
CanReadData The user can view mapped data, in other words SDMX data for the Dataflows that he has access to.
CanReadStructuralMetadata The user can view and interact (view details, download) with the Browse & Download screen (list structures, view details, download). This permission is also needed to import structures from Import from File & Import from Registry screens, as well as view the Category Schemes/Categories/Dataflows in the Category Management.
CanUpdateData The user can update data in dissemination database (not impemented yet)
CanUpdateStructuralMetadata The user can update structural metadata (used by OECD)

THE FOLLOWING ROLES ARE AVAILABLE IN MAWEB

Role Permissions Description
AdminRole CanUpdateData, CanUpdateStructuralMetadata, DataImporterRole, DomainUserRole, StructureImporterRole, WsUserRole Administrator
DomainUserRole CanIgnoreProductionFlag, CanPerformInternalMappingConfig, WsUserRole Mapping Assistant user
WsUserRole CanReadData, CanReadStructuralMetadata NSI Web Service user
StructureImporterRole CanDeleteStructuralMetadata, CanImportStructures, CanReadStructuralMetadata, CanUpdateStructuralMetadata NSIWS User who can submit sturctural metadata
