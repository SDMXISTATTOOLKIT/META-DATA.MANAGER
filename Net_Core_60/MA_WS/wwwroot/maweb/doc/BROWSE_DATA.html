BROWSE AND DOWNLOAD DATA

In this screen, the user can browse and preview or download data by selecting a Dataflow from the left hand side. On the right hand side we have some options that affect the content of the data file that will be retrieved from the server. First of all, the user can choose the data format. The available options are the following:

  • SDMX-ML 2.0 Compact
  • SDMX-ML 2.0 Cross Sectional
  • SDMX-ML 2.0 Generic
  • GESMESTS
  • SDMX-ML 2.1 Generic
  • SDMX-ML 2.1 Structure Specific
  • CSV

By clicking "Download Data", an .xml file should be downloaded after the loading screen. The name of the file should contain the ID, AGENCY, VERSION of the Dataflow.

DATA PREVIEW

By clicking "Preview Data", a new window should open. It will contain the first 10 results of the requested data inside a text area. This is only for preview purposes, and with "Copy to clipboard" the server should return all of the data which is automatically copied to clipboard. Then, the user can paste the data somewhere else.
