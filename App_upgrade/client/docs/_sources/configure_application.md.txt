# How to configure the application

To manage the Configuration, the System provides an application interface which can be used by a profiled user with **SuperUser** permissions, also referred to in the guide as [**superadmin**](superadmin.md). \
The reference function can be found in the left-hand side menu under **Configuration**.  \
The item of interest is **_Application_**. \
After entering the credentials of [**superadmin**](superadmin.md), the System opens the management mask of the application configuration by positioning itself in the section "**User Interface**". \
The following configurations can be made for the Data and Meta Manager applications:
